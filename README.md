# Prerequisites

First install perlbrew (http://perlbrew.pl and a recent perl).

Then, `perlbrew install-cpanm` to install cpanminus.

Install prerequisites:

    cpanm --installdeps .

# Run the tests

    ./ata test

# Run the server

    ./ata daemon
